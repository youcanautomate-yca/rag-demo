
# 🚀 RAG Demo - Retrieval Augmented Generation Project

## 📋 Overview

This is a comprehensive **Retrieval Augmented Generation (RAG)** project that demonstrates how to build an intelligent document-based chatbot. The project covers the complete RAG pipeline - from data ingestion and preprocessing to embedding generation and vector database storage using **Qdrant** and **Ollama**.

## 🎯 What is RAG?

RAG (Retrieval Augmented Generation) is a powerful technique that combines:
- **Retrieval**: Extract relevant information from documents using semantic search
- **Augmentation**: Enhance prompts with retrieved context
- **Generation**: Use LLMs to generate accurate answers based on retrieved data

This approach ensures your AI model provides accurate, context-aware responses grounded in your actual documents.

## ✨ Project Features

- 📄 **PDF Processing**: Load and extract text from PDF documents
- 🔍 **Text Preprocessing**: Clean and normalize document content
- ✂️ **Intelligent Chunking**: Split documents into manageable chunks
- 🧠 **Embedding Generation**: Create vector embeddings using Ollama
- 💾 **Vector Storage**: Store embeddings in Qdrant vector database
- 🤖 **Smart Querying**: Retrieve relevant context and generate answers
- 🗣️ **Local LLM**: Run everything locally using Ollama
